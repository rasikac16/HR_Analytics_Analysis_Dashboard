# HR_Analytics_Analysis_Dashboard
## Project Overview
This project uses the HR Analytics dataset to analyse workforce demographics, employee attrition, and departmental performance. Using Microsoft Excel, the goal is to identify patterns influencing employee retention, evaluate workforce distribution, and support HR in improving engagement and decision-making through data insights.
________________________________________
## Dataset Description
The dataset contains detailed employee information, including:
•	Personal Details: Employee ID, Age, Gender, Age Group.
•	Job Information: Department, Job Role, Business Travel, Distance from Home.
•	Compensation & Education: Daily Rate, Education, Education Field.
•	Attrition Data: Attrition status (Yes/No).
This dataset provides a comprehensive view of workforce dynamics and attrition trends across multiple HR variables.
_______________________________________
## Business Objectives
The main objectives of this HR Analytics project are:
1.	Identify factors influencing employee attrition.
2.	Analyse department-wise employee distribution.
3.	Compare attrition trends across age groups, education fields, and job roles.
4.	Measure gender diversity and its relationship with attrition.
5.	Build an interactive HR dashboard to help management explore employee data dynamically.

## Key Insights
•	Younger employees (20–30 years) showed higher attrition rates.
•	Sales and R&D departments had the highest turnover rates.
•	Males showed slightly higher attrition compared to females.
•	Employees with higher education tended to stay longer in the organization.
•	Business travel frequency was a major factor linked to attrition.
